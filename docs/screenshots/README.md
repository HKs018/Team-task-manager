# Screenshot Guide

Save README screenshots in this folder using these exact names:

1. `01-login.png` - Login/signup screen before logging in.
2. `02-dashboard.png` - Admin dashboard with stats, recent projects, and active tasks.
3. `03-projects.png` - Projects page with project list, selected project, members, and task board.
4. `04-tasks.png` - Tasks page showing all visible tasks.
5. `05-member-view.png` - Member account view showing that admin-only controls are hidden.
6. `06-mobile.png` - Narrow/mobile layout.

Best size: full browser window screenshot on laptop, then one narrow/mobile screenshot.
